# 🛍️ DSCommerce Layout

Projeto de interface desenvolvido em **HTML** e **CSS** com base nos protótipos do sistema **DSCommerce**.  
O objetivo deste repositório é representar visualmente as principais telas da aplicação, simulando a experiência de navegação de um e-commerce com área pública e área administrativa.

Este projeto foi construído a partir do documento de requisitos do **DSCommerce**, utilizado nos cursos da DevSuperior, e tem foco na prática de estruturação de páginas, organização visual e construção de layouts para sistemas web.

## 📖 Sobre o projeto

O sistema **DSCommerce** foi concebido para simular um e-commerce com funcionalidades como:

- catálogo de produtos
- visualização de detalhes do produto
- carrinho de compras
- login e cadastro
- confirmação de pedido
- área administrativa para manutenção de dados

Neste repositório, o foco está na **implementação do layout das telas**, servindo como base visual para o sistema e como prática de desenvolvimento front-end com HTML e CSS.

## 🎯 Objetivo

Desenvolver as interfaces do sistema DSCommerce a partir dos protótipos e requisitos funcionais, praticando:

- estruturação semântica com HTML
- estilização com CSS
- organização de arquivos em um projeto front-end
- construção visual de telas inspiradas em um sistema real
- separação entre área pública e área administrativa

## 🖥️ Telas desenvolvidas

O projeto contém as seguintes páginas:

- `login.html` — tela de login
- `catalogo.html` — catálogo de produtos
- `detalhes.html` — detalhes do produto
- `carrinho.html` — carrinho de compras
- `confirmacao.html` — confirmação de pedido
- `listagem.html` — listagem administrativa
- `form.html` — formulário administrativo
- `admin-home.html` — página inicial da área administrativa
